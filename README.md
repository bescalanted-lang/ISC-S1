# Tarea Semana 1
Glorsario
## **1.Phishing**: 
Es una tecnica de ingenieria social y ciberdelito que busca engañar a usuarios para robar información confidencial como credenciales, datos bancarios, números de tarjeta, etc.
## **2.Ley de Moore**:
Es una observación formulado por Gordon Moore en 1965 que establece que el número de transistores de un microchip se duplica aproximadamente cada dos años. 
## **3.Hardware**:
Se refiere a todos los componentes físicos, tangibles y materiales que constituyen un sistema informatico o computadora.
## **4.Software**:
Es el conjunto de componentes lógicos, programas, instrucciones y reglas informáticas que permiten que dispositivos físicos realizar tareas específicas.
## **5.UEFI**:
Significa "Interfaz de firmware extensible unificada" que remplaza a la BIOS y actua como la primera capa de software que se ejucuta al arrancar el equipo.
## **6.APIs**:
Significa "Interfaces de programacion de aplicaciones", son conjuntos de reglas que permiten que diferentes aplicaciones de software compartan datos entre sí.
## **7.Docker**:
Es una plataforma de código abierto que permite empaquetar aplicaciones y sus dependencias en "contenedores" ligeros y portatiles.
## **8.DevOps**:
Es un conjunto de prácticas y herramientas que integra a los equipos de desarrollo para optimizar la calidad, velocidad y seguridad en la entrega de software.
## **9.loT**:
Generalmente se refiere al internet de las cosas que son sistemas de dispositivos físicos conectadosa redes inalambricas que reciben y transfieren datos sin intervencion humana.
## **10.Microcontroladores**:
Es un circuito integrado (chip) que funciona como una pequeña computadora en un solo componente, integrando CPU, memoria (RAM y ROM) y puertos de entrada/salida.  
## **11.Firmware**:
Es un software de bajo nivel, integrado directamente en los circuitos electrónicos que actua como puente escencial entre el hardware y el sistema operativo.
## **12.Transistor**:
Es un componente electrónico semiconductor fundamental, utilizado para amplificar o conmutar señales y energía eléctrica.
## **13.Periféricos**:
Son dispositivos de hardware externos o independientes que se conectan a una computadora para ampliar sus funcionalidades, permitiendo la entrada y salida.
## **14.Sistema Operativo**:
Es el software fundamental que gestiona el hardware y permite la ejecución de aplicaciones en computadoras, teléfonos inteligentes y otros dispositivos.
## **15.IaaS**:
Es un modelo de computación en la nube que ofrece recursos esenciales de TI (servidores, almacenamiento y redes) bajo demanda a través de internet.
