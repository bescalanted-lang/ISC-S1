# Tarea Semana 1
Glorsario
## **1.Phishing**: 
Es una tecnica de ingenieria social y ciberdelito que busca engañar a usuarios para robar información confidencial como credenciales, datos bancarios, números de tarjeta, etc. Suplantando la entidad de entidades de confianza (bancos, empresas, conocidos) mediante correos electronicos, SMS, o sitios web falsos.
## **2.Ley de Moore**:
Es una observación formulado por Gordon Moore en 1965 que establece que el número de transistores de un microchip se duplica aproximadamente cada dos años, aumentando la potencia de procesamiento y reduciendo costos. 
## **3.Hardware**:
Se refiere a todos los componentes físicos, tangibles y materiales que constituyen un sistema informatico o computadora, incluyendo partes eléctricas, eléctronicas y electromecánicas. Abarca dispositivos internos como el procesador (CPU), memoria RAM, placa base y almacenamiento, así como periféricos de entrada/salida.
## **4.Software**:
Es el conjunto de componentes lógicos, programas, instrucciones y reglas informáticas que permiten que dispositivos físicos (hardware) realizar tareas específicas. Es la parte intangible de los equipos, fundamental para su funcionamiento y la interacción del usuario.
## **5.UEFI (Unified Extensible Firmware Interface)**:
Es un estandar de firmware moderno que remplaza a la BIOS tradicional de los ordenadores, actuando como la primera capa de software que se ejucuta al arrancar el equipo. Ofrece ventajas clave como tiempos de inicio más rápidos, seguridad mejorada como secure boot, soporte para discos duros de mas de 2.2 TB.
## **6.APIs (Application Programming Interfaces)**:
Son conjuntos de reglas y protocolos que permiten que diferentes aplicaciones de software se comuniquen y compartan datos entre sí, actuando como intermediarios. Son fundamentales para integrar servicios, automatizar flujos de trabajo y facilitar el desarrollo de sooftware moderno sin necesidad de programar desde cero.
## **7.Docker (Contenedores)**:
Es una plataforma de código abierto que permite empaquetar aplicaciones y sus dependencias en "contenedores" ligeros y portatiles, garantizando que funcionen igual en cualquier computadora, ya sea local o en la nube. Facilita el desarrollo, prueba y despliegue rápido al aislar aplicaciones.
## **8.DevOps**:
Es un conjunto de prácticas, herramientas y una filosofía cultural que integra a los equipos de desarrollo y operaciones para optimizar la calidad, velocidad y seguridad en la entrega de software. Se basa en la colaboración, la automatización del ciclo de vida y la mejora continua, eliminando barreras entre equipos para ofrecer valor constante a los clientes.
## **9.loT**:
Generalmente se refiere al internet de las cosas que son sistemas de dispositivos físicos conectadosa redes inalambricas que reciben y transfieren datos sin intervencion humana.
## **10.Microcontroladores**:
Es un circuito integrado (chip) que funciona como una pequeña computadora en un solo componente, integrando CPU, memoria (RAM y ROM) y puertos de entrada/salida, diseñado para ejecutar tareas específicas en sistemas embebidos, como controlar electrodomésticos, sistemas automotrices o dispositivos IoT, siendo programable y muy eficiente en consumo energético.  
## **11.Firmware**:
Es un software de bajo nivel, integrado directamente en los circuitos electrónicos (memoria ROM o flash) de un dispositivo, que actua como puente escencial entre el hardware y el sistema operativo.
## **12.Transistor**:
Es un componente electrónico semiconductor fundamental, utilizado para amplificar o conmutar señales y energía eléctrica. Actúa como interruptor o amplificador, controlando el flujo de corriente entre sus terminales (base, colector, emisor) mediante una pequeña tensión de control.
## **13.Periféricos**:
Son dispositivos de hardware externos o independientes que se conectan a una computadora para ampliar sus funcionalidades, permitiendo la entrada, salida, almacenamiento o comunicación de datos. 
## **14.Sistema Operativo**:
Es el software fundamental que gestiona el hardware y permite la ejecución de aplicaciones en computadoras, teléfonos inteligentes y otros dispositivos. Actúa como un intermediario entre el usuario y los componentes físicos (CPU, memoria, almacenamiento), facilitando la interacción y coordinando recursos de manera eficiente. 
## **15.IaaS**:
Es un modelo de computación en la nube que ofrece recursos esenciales de TI (servidores, almacenamiento y redes) bajo demanda a través de Internet. Permite a empresas alquilar esta infraestructura a proveedores externos (como Google Cloud, AWS, Azure) bajo un modelo de pago por uso, eliminando la necesidad de gestionar centros de datos físicos, reduciendo costos y escalando recursos ágilmente.
